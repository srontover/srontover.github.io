## 项目经历

### T1 人形机器人踢球强化学习任务开发

我基于 **MJLab、MuJoCo 和 Warp** 构建了 T1 人形机器人踢球强化学习任务，完成了从环境配置、动作空间设计、观测建模、奖励函数、终止条件到课程学习的完整训练流程。

该任务的核心目标是让人形机器人在保持身体稳定的同时，能够根据球的位置和目标方向完成有效踢球。相比普通 locomotion 任务，踢球任务对机器人提出了更复杂的控制要求：机器人不仅要站稳，还要选择合适的支撑脚与踢球脚，在合适的时间完成出脚动作，并让球获得足够速度和正确方向。

在该项目中，我主要完成了以下工作：

* 搭建 T1 humanoid 踢球强化学习环境，配置机器人动作空间、观测空间、球体 reset 逻辑、接触传感器、奖励项、终止条件和训练日志指标。
* 设计机器人下半身及腰部、头部相关动作控制，使机器人能够在保持平衡的基础上完成面向球、调整姿态和出脚动作。
* 建立脚与球之间的接触检测机制，用于判断有效触球、踢球脚选择和接触质量。
* 设计分阶段 reward shaping 策略，使机器人从固定正前方轻触球逐步过渡到随机目标方向和距离下的稳定踢球。
* 引入课程学习机制，逐步增加任务难度，包括球的位置随机化、目标方向随机化、目标距离变化和视觉误差扰动。
* 通过训练日志分析机器人策略的学习过程，包括触球率、球速、球前进距离、目标方向误差、站立稳定性和 episode termination 原因等指标。
* 针对训练中出现的“只靠近球但不出脚”“踢球脚选择不稳定”“触球后球速不足”“身体失衡提前终止”等问题，迭代调整 reward 权重和训练阶段。

该项目让我系统理解了机器人强化学习任务从环境建模到策略训练的完整流程，也让我认识到复杂机器人任务中 reward design、curriculum learning 和 observation design 对最终策略质量的决定性影响。

### 人形机器人 locomotion 策略训练

在踢球任务之前，我已经完成了机器人基础 locomotion 策略训练，使机器人能够在仿真环境中实现稳定行走和速度跟踪。该部分工作为后续踢球任务提供了基础运动能力，也帮助我理解机器人强化学习中姿态稳定、速度跟踪、能耗控制和动作平滑之间的权衡。

在该项目中，我重点关注：

* 机器人状态观测设计，包括关节位置、关节速度、身体姿态、角速度、速度命令等信息。
* locomotion reward 设计，包括速度跟踪、姿态稳定、能耗惩罚、动作平滑、脚部接触和身体高度约束。
* 训练过程中不同 reward 项之间的竞争关系，以及某些奖励过强时对策略行为的影响。
* 从基础 locomotion 到更复杂任务的迁移，例如将行走能力与踢球动作结合。

这部分经历让我意识到，复杂机器人行为通常不能只依赖单一任务奖励直接端到端训练，而需要合理拆分阶段、设计中间能力，并在不同训练目标之间保持平衡。

### 基于运动数据的人形机器人动作先验探索

我对使用人类运动数据构建机器人动作先验也有较强兴趣，尤其关注 **AMASS 数据、AMP、encoder-decoder 架构、motion latent** 等方法在机器人控制中的应用。

在相关探索中，我尝试理解和分析如何利用人类运动数据为机器人提供更自然、更稳定的动作分布约束。相比直接通过 reward 手工设计所有动作细节，动作先验可以帮助机器人在训练早期避免大量不自然动作，并提升策略学习效率。

我重点关注的问题包括：

* 如何从人类运动数据中提取对机器人有用的运动模式。
* 如何通过 AMP 或类似判别器方法约束机器人动作风格。
* 如何使用历史帧、encoder、policy 和 decoder 架构压缩运动信息并输出控制动作。
* 如何将 locomotion、get-up 和 kicking 等不同技能统一到更连续的运动表示中。
* 如何在任务奖励和动作先验之间设置合适权重，避免先验过强导致任务完成能力下降，或先验过弱导致动作质量不足。

这一方向也是我未来希望继续深入研究的内容。我认为机器人强化学习中的动作先验不仅是“让动作更像人”，更重要的是为高维连续控制问题提供更好的策略搜索空间，使机器人能够在复杂任务中更快学习到稳定、可解释、可迁移的行为。
